# Guia de Suporte TI

Aplicação desktop desenvolvida para auxiliar usuários finais
com orientações simples, procedimentos e soluções rápidas
para problemas comuns de TI, sem a necessidade de abrir chamados.

Além disso, o aplicativo atua como um canal direto de comunicação
entre o time de TI e os colaboradores, permitindo a divulgação
de avisos, boas práticas, orientações preventivas e comunicados internos.

## Objetivo
Centralizar informações técnicas e operacionais em um único app
offline, promovendo autonomia dos usuários, fortalecendo a
comunicação com a TI e reduzindo a demanda de chamados.

## Benefícios
- Redução de chamados repetitivos
- Comunicação direta entre TI e colaboradores
- Disseminação de boas práticas de segurança e uso de sistemas
- Agilidade na resolução de problemas simples
- Melhoria na experiência do usuário
- Otimização do tempo do time de suporte

## Tecnologias
- HTML
- CSS
- JavaScript
- Electron (Runtime Desktop)
