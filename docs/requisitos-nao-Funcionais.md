RNF01 - A aplicação deve funcionar 100% offline.

RNF02 - Deve apresentar baixo consumo de memória e processamento.

RNF03 - Deve ser compatível com máquinas de desempenho reduzido.

RNF04 - Não deve utilizar frameworks externos.

RNF05 - Não deve armazenar dados sensíveis.

RNF06 - Deve possuir interface intuitiva e de fácil navegação.

RNF07 - Deve permitir futura integração com APIs externas sem reestruturação completa.
