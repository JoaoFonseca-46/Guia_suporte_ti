O Guia de Suporte - TI é uma aplicação desktop informativa
voltada ao autoatendimento de colaboradores para resolução
de problemas simples de tecnologia (nível N1).

A ferramenta está sendo desenvolvida para:

- Reduzir chamados recorrentes de baixa complexidade
- Padronizar orientações fornecidas pelo suporte
- Melhorar a comunicação entre TI e usuários
- Promover boas práticas no uso de recursos tecnológicos

A aplicação possui caráter exclusivamente informativo,
não armazenando dados sensíveis nem executando operações administrativas.

O sistema foi projetado para funcionar offline,
com baixo consumo de recursos e compatibilidade com diferentes unidades e configurações de TI.
